# Tutorial: Creating a circle

**Runtime:** 75 seconds (target ~185 spoken words)
**Goal:** First-time user can create their first circle, set the
contribution amount and member count, and send invites.

---

## Scene 1 -- Hook (0:00 -- 0:08)

- **Scene:** Cold open on a phone, hand tapping the Circles tab in the
  TandaXn bottom nav. Soft motion blur on the swipe.
- **Narration:** "Saving with people you trust is the oldest financial
  tool there is. TandaXn calls them circles."
- **On-screen:** `Circles. Save together.`

## Scene 2 -- Start the flow (0:08 -- 0:20)

- **Scene:** The Circles tab is open. A bright `+` floating action
  button pulses on the bottom-right; finger taps it. Screen transitions
  to Create Circle.
- **Narration:** "Tap the plus button on your Circles tab to start a
  new one. You'll pick a type, an amount, and who to invite."
- **On-screen:** Arrow pointing at `+ button` -> caption `Step 1: Tap +`

## Scene 3 -- Pick a type (0:20 -- 0:35)

- **Scene:** Circle Type picker: cards for Traditional, Goal, Emergency,
  Investment, Charity. Finger lingers on Traditional, then taps it.
- **Narration:** "Choose a type. Traditional rotates payouts evenly,
  Goal is for a shared target, Emergency builds a safety net. Pick the
  one that matches what you're saving for."
- **On-screen:** `5 circle types -- pick the fit`

## Scene 4 -- Set amount + members (0:35 -- 0:55)

- **Scene:** Form with two inputs: contribution amount (slider /
  number) set to $100, member count (stepper) set to 6. Cycle
  frequency dropdown shows Weekly. Live preview card on the right shows
  payout = $600 per cycle.
- **Narration:** "Set the contribution each person pays per cycle and
  how many members will join. The preview shows what each payout will
  be -- everyone gets a turn."
- **On-screen:** `$100 / week x 6 members = $600 payout per cycle`

## Scene 5 -- Invite + done (0:55 -- 1:15)

- **Scene:** Invite screen -- contacts list with checkmarks next to a
  few selected names, plus a Share Link button. Finger taps Share Link;
  a system share sheet animates up. Then a green success screen with
  the new circle card.
- **Narration:** "Invite people by phone, email, or a shareable link.
  Your circle becomes active the moment everyone confirms."
- **On-screen:** `Invite -> Confirm -> Start contributing`

## End CTA (1:15 -- 1:15)

- **Scene:** End card -- TandaXn logo, copy below.
- **On-screen:**
  ```
  Ready to start saving together?
  Open the Circles tab and tap +.
  ```
